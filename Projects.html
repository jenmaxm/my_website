Daisyworld Model

For my A-Level project, I developed a user-friendly graphical user interface (GUI) to better visualize and understand the Daisyworld modelâ€”a theoretical simulation that explores the intricate feedback mechanisms between life and its environment.

The Daisyworld model, originally proposed by James Lovelock, demonstrates how simple organisms, in this case, black and white daisies, can influence and regulate planetary temperature through their impact on albedo (the reflectivity of a surface).
By absorbing or reflecting solar radiation, these daisies create a dynamic, self-regulating climate system that maintains equilibrium under varying solar luminosity.

My project aimed to break down this concept into a more accessible and interactive format, allowing users to manipulate key variables such as solar input, daisy growth rates, and environmental conditions, observing in real-time how these factors influence planetary temperature and ecosystem stability.
The GUI provides an intuitive way to visualize complex interactions, making it easier to grasp the profound implications of biotic-abiotic feedback loops in climate regulation.

This project not only deepened my understanding of computational modeling and system dynamics but also highlighted the significance of self-regulating mechanisms in planetary ecosystems.

Online shop

This project is a Java-based online shopping system with a graphical user interface (GUI) that allows users to browse, add products to a cart, and proceed to checkout. The system is designed to handle product management, user interaction, and a simulated checkout experience.

It utilizes Java Swing or JavaFX for building the GUI, object-oriented programming (OOP) for structuring products, user roles, and shopping logic, and event-driven programming for handling user interactions through buttons, drop-down menus, and form inputs.

Weather App

This Python script creates a fully functional Weather App using the tkinter library for the graphical user interface (GUI) and the OpenWeather API to retrieve real-time weather data.

The app provides an intuitive and user-friendly interface where users can input the name of any city, triggering an API request to fetch comprehensive weather details, including temperature, weather conditions, and atmospheric descriptions.

The temperature is displayed in Celsius for easy interpretation, and a corresponding weather icon is dynamically retrieved and displayed, enhancing the visual appeal of the application.

This project not only demonstrates the integration of APIs with Python but also showcases how GUI-based applications can provide real-time, interactive experiences for users.

Music Player

This Python script creates an intuitive and user-friendly Music Player application, utilizing the Tkinter library for the graphical user interface (GUI) and the Pygame library for seamless music playback.

The application allows users to load MP3 files from a selected folder, providing essential playback controls such as play, pause, resume, and stop.

Additionally, users can easily navigate through their music collection, selecting and playing different tracks effortlessly.

With a simple yet functional design, this project showcases the integration of Python libraries to create interactive and engaging multimedia applications.
